História dos computadores

O computador, tal qual conhecemos hoje, passou por diversas transformações e foi se aperfeiçoando ao longo do tempo.

A história do computador pode ser divida, até o momento, em cinco gerações:

1º Geração (1946 - 1956)

  • Tinha como principal característica o uso de válvulas eletrônicas
  • Possuiam o uso restrito, além de serem imensos e consumirem muita energia
  • Em 1946 ocorreu o lançamento do ENIAC Sua principal inovação foi a computação digital, ou seja, a entrada de dados era realizada no painel de controle e cada operação podia ser acessada através de configurações-padrão de chaves e switches.


    ENIAC - Electrical Numerical Integrator and Calculator

2º Geração (1957 - 1964)

  • Tinha como principal característica o uso de transistores
  • Em janeiro de 1959 a Texas Instruments anuncia ao mundo uma criação de Jack Kilby: o circuito integrado.

3º Geração (1965 - 1970)

  • Caracterizada pela introdução dos circuitos integrados
  • Criação de mini-computadores
  • Surgem conceitos como memória virtual, multiprogramação e sistemas operacionais complexos

4º Geração (1971 - ao momento)

  • Caracterizada pela introdução dos microprocessadores

  • Desenvolvimento dos computadores pessoais

5º Geração (o presente e o futuro)

  • Baseia-se principalmente na inteligência artificial e seus diversos ramos:
    • Redes Neurais
    • Sistemas Inteligentes
    • Reconhecimento de Voz

Comunicação Client x Server

Surgimento da Internet

A Internet surgiu a partir de pesquisas militares no auge da Guerra Fria.

Arquitetura Client x Server

A comunicação na Internet é baseada no modelo cliente-servidor que faz uso de protocolos de comunicação do tipo requisição/resposta.

É o Client/Cliente quem sempre inicia a comunicação com o servidor e seu representante mais conhecido é o navegador

O papel do Server/Servidor é receber uma requisição e devolver uma resposta para o cliente.




Nos servidores podem ser hospedados documentos, imagens, áudio, vídeo, etc., assim como, programas que atendem as requisições dos clientes e as processam.

Existem diversos tipos de servidores para atender as requisições do cliente.

São exemplos de tipos de servidores:

  • Arquivos
  • Segurança (Firewall)
  • Streaming
  • E-mail
  • Web

Os servidores possuem hardwares e softwares voltados para atender as requisições e servidores Web geralmente são capazes que executar aplicações em linguagens como PHP, Ruby, JS, Java, C#, dentre outras.

A linguagem JavaScript pode ser executada tanto no lado do cliente quanto do lado do servidor.

No lado cliente, o JavaScript é executado pelo navegador. Para execução e interpretação de JavaScript, geralmente o navegador utiliza o módulo v8

No lado servidor, o JavaScript é executado pelo Node.js

HTML na Prática

Estrutura Básica

Por convenção, a página inicial de um projeto HTML é a página index.html.

A estrutura básica de uma página HTML é composta pelas tags:

  • html
  • head
  • body

Listas

Listas são muito utilizadas quando queremos listar itens no site e também para a criação de menus de navegação.

No HTML temos três tipos de listas:

  • Listas Ordenadas

    • Utiliza a tag <ol> e os respectivos itens da lista ficam dentro da tag <li>.

    Por padrão são apresentadas no navegador ordenadas por números, podendo alterar este tipo da marcação utilizando-se o atributo type


  • Listas Não-Ordenadas

    • Utiliza a tag <ul> e os respectivos itens da lista também ficam dentro da tag <li>.

    Por padrão são representadas no navegador por "bullets", podendo alterar este tipo da marcação via CSS.


  • Listas de Definições

    • As listas de definições são representadas pela tag <dl>, e cada termo da lista de definição é representado pelo par de tags <dt> e <dd>

Links

A tag <a> define um hiperlink, que é usado para vincular uma página a outra.

O atributo mais importante da tag <a> é o atributo href, que indica o destino do link.

Outros atributos relevantes da tag <a> são o atributo target que indica se o link será exibido na janela atual do navegador (padrão) ou se será apresentado em outra aba e o atributo title que representa um tooltip do link
